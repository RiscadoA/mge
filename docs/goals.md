# Goals

## Platforms

The following platforms will be supported (in order of priority)

- Windows (desktop and VR).
- Linux (desktop and VR).
- Mac OSX (desktop and VR).

## Animation

- Inverse kinematics.
- Playing skeletal animation files.

## Rendering

- Both VR and window rendering.
- Deferred rendering.
- Transparency.
- Swappable backend.
- Particle systems.
- Static meshes.
- Skeleton meshes (skins).
- Mesh instancing.
- Post-rendering full screen effects (SSAO).
- 2D GUI elements.
